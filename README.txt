# Multi-page Academic Site

## Files
- `index.html` — landing with short bio + Scholar/ORCID.
- `about.html` — full About + Research interests.
- `publications.html` — publications list (edit this when adding new papers).
- `teaching.html` — teaching info (edit later).
- `mylab.html` — lab page (edit later).
- `cv.html` — CV download link (add `cv.pdf` alongside).
- `avatar.jpg` — replace with your real portrait.

## Publish on GitHub Pages
1) Create repo `USERNAME.github.io` on GitHub.
2) Upload all these files (and `cv.pdf` if you have).
3) Settings → Pages → (main / root). Then open https://USERNAME.github.io
